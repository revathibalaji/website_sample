history of FixNix
     Fix Nix is an Information Security company born on a vision to democratize the security landscape of the world. Fix Nix provides varied services in the security domain from a Security Audit to a Virtual Chief Information Security Officer (VCISO).
      Businesses now run on the computer and are empowered by the cloud. So it is evident that Information Security and Data Privacy has emerged as a major concern among all stake holders from Governments, Business Entities to the end customer. We sensed there is a need for every Small and Medium Enterprise (SME) in the world today for an information security product, at least a data encryption and a data loss prevention module. There is virtually no one catering to their needs. Every Company in the ISMS and GRC (Governance, Risk Management, Compliance) space offers services and management systems only to the Fortune 1000 companies of the world. So was born FIXNIX. We don’t just fix the security issues but we Nix it through an affordable, feasible, agile, cloud based Management system in “Security as a Service model”
